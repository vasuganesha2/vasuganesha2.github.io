EXERCISE 3.37

Important Problems: Combinatorics

Advanced Set Theory & Inequality Proofs

Set Theory Inequalities Indicator Variables
There are \(N\) students at Grant High School. Let \(S(F)\) be the number of students who speak French, \(S(J)\) be the number who speak Japanese, and \(S(A)\) be the number who speak Arabic. Let \(S(AF)\) be the number who speak both French and Arabic; define \(S(AJ)\) and \(S(FJ)\) similarly.

Prove that: \[ 3N + S(AF) + S(AJ) + S(FJ) \ge 2S(A) + 2S(F) + 2S(J) \]

Proof via Indicator Variables

For each student \(x\), define the indicator variables \(a_x, f_x, j_x \in \{0, 1\}\): \[ a_x = \begin{cases} 1 & x \in A \\ 0 & x \notin A \end{cases}, \quad f_x = \begin{cases} 1 & x \in F \\ 0 & x \notin F \end{cases}, \quad j_x = \begin{cases} 1 & x \in J \\ 0 & x \notin J \end{cases} \]
The set sizes can be expressed as sums over all students \(x \in \{1, \dots, N\}\): \[ S(A) = \sum a_x, \quad S(AF) = \sum a_xf_x, \quad \dots \] The inequality holds if it holds for every individual student \(x\): \[ 3 + a_xf_x + a_xj_x + f_xj_x \ge 2(a_x + f_x + j_x) \]

Verification

Let \(k = a_x + f_x + j_x\) be the number of languages student \(x\) speaks.

Case (\(k\)) LHS: \(3 + \sum \text{products}\) RHS: \(2k\) Result
\(k = 0\) \(3 + 0 = 3\) \(2(0) = 0\) \(3 \ge 0\) ✓
\(k = 1\) \(3 + 0 = 3\) \(2(1) = 2\) \(3 \ge 2\) ✓
\(k = 2\) \(3 + 1 = 4\) \(2(2) = 4\) \(4 \ge 4\) ✓
\(k = 3\) \(3 + 3 = 6\) \(2(3) = 6\) \(6 \ge 6\) ✓
Alternative Non-negative Expression:
Observe that \((1-a_x)(1-f_x) + (1-a_x)(1-j_x) + (1-f_x)(1-j_x) \ge 0\) because each term is a product of non-negative integers. Expanding this yields the desired local inequality.
Hence proved. ■
